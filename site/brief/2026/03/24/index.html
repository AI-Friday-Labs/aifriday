Monday, March 24, 2026

Sora is dead. OpenAI's video generation product announced its shutdown today, which didn't stop OpenAI from also unveiling shopping inside ChatGPT via something called the Agentic Commerce Protocol. Meanwhile, Anthropic shipped auto mode for Claude Code, a safer alternative to --dangerously-skip-permissions that uses a separate classifier model to gate actions. And three solid Show HNs today, including sub-second video search built on Gemini's new native video embeddings.

Big Moves

Goodbye to Sora

OpenAI is shutting down Sora, their video generation tool. 987 points on HN, which tells you people had feelings about this one. Sora launched to enormous hype, and now it's gone. The video generation space has gotten crowded and Sora never really found its footing after the initial demo reel.
Hacker News

Auto Mode for Claude Code

Anthropic introduced auto mode, a new permissions mode in Claude Code. Instead of approving every action or using --dangerously-skip-permissions, Claude now makes permission decisions on your behalf with a safety net: a separate Claude Sonnet 4.6 classifier reviews each action before it runs. Simon Willison has the breakdown. This is a good middle ground — the constant permission prompts were the main reason people reached for the dangerous flag in the first place.
Simon Willison

OpenAI Puts Shopping in ChatGPT

ChatGPT now does product discovery and side-by-side comparisons, powered by something OpenAI is calling the Agentic Commerce Protocol. Merchants can integrate directly. So ChatGPT is a shopping assistant now. The name is a lot.

Running Models Locally

Streaming Experts: 1 Trillion Parameters in 96GB of RAM

Simon Willison wrote up the latest on streaming MoE expert weights from SSD. Five days ago Dan Woods was running Qwen3.5-397B in 48GB of RAM. Now @seikixtc is running Kimi K2.5 — a 1 trillion parameter model with 32B active weights — in 96GB of RAM on a Mac. The trick: stream only the expert weights you need for each token from SSD instead of holding the whole model in memory. Slow, but it works, and it's getting faster.
Simon Willison

Hypura — LLM inference scheduler for Apple Silicon

A storage-tier-aware inference scheduler built specifically for Apple Silicon's unified memory architecture. 213 points on HN. If you're running models on a Mac, this is worth a look — it's aware of the difference between memory and SSD tiers and schedules accordingly.
Hacker News

Tooling & Infrastructure

FastMCP

FastMCP is a framework for building MCP servers. Clean getting-started docs. If you've been meaning to build an MCP server and wanted something more structured than rolling your own, this is the on-ramp.

Sandboxing AI Agents, 100x Faster

Cloudflare announced Dynamic Workers for sandboxing AI agent code execution. The pitch: spin up an isolated environment for each agent action, 100x faster than container-based approaches. As agents get more autonomy (see: Claude Code auto mode above), sandboxing matters more.

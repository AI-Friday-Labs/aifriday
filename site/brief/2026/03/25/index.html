Tuesday, March 25, 2026

The stack is maturing fast — music generation APIs, smarter coding agents, and a whole wave of tools making local inference actually practical. Less hype, more shipping.

🎵

Google launches Lyria 3 — music generation via the Gemini API

Google's newest music generation model is now available in paid preview through the Gemini API and testable in AI Studio. Lyria 3 Pro brings longer tracks and tighter integration into professional creation tools.

Music gen just got a proper API. If you're building anything with audio, this is your on-ramp.

🤖

Auto mode for Claude Code

Claude Code now has an 'auto mode' where Claude makes permission decisions on your behalf — with a separate Sonnet 4.6 classifier reviewing each action before it runs. It's the middle ground between full manual approval and `--dangerously-skip-permissions`.

A real step toward coding agents you can trust to run without babysitting every shell command.

🗜️

TurboQuant: Google's extreme model compression

New research from Google pushes quantization to its limits, dramatically shrinking model sizes while keeping quality surprisingly intact. This is the kind of work that makes bigger models fit on smaller hardware.

If you care about running models locally or cutting inference costs, this is the research to watch.

🔍

SentrySearch — sub-second video search powered by Gemini's native video embeddings

A Show HN project that takes advantage of Gemini's new native video embedding capability to build blazing-fast semantic video search. Point it at your footage and query naturally.

Gemini can natively embed video now — and this is a clean demo of what you can build with it.

🍎

Hypura — storage-tier-aware LLM inference for Apple Silicon

An open-source scheduler that understands your Mac's memory hierarchy (unified memory, SSD swap, etc.) to run LLM inference more efficiently. Designed specifically for Apple Silicon's quirks.

Finally, someone built inference tooling that actually respects how Apple Silicon manages memory.

⚡

FastMCP — the fast way to build MCP servers

A clean, batteries-included framework for building Model Context Protocol servers. If you've been wanting to expose your tools and data to AI agents via MCP, this smooths out the rough edges.

MCP is becoming the standard agent-to-tool protocol. FastMCP makes it easy to actually ship one.

👁️

ProofShot — give AI coding agents eyes to verify their own UI

A neat Show HN project that screenshots the UI an AI agent just built and feeds it back for visual verification. Closes the loop between 'generate code' and 'does this actually look right.'

Coding agents are great at generating UI — and terrible at checking it. This fixes the feedback loop.

📦

Cloudflare Dynamic Workers — sandbox AI agents 100x faster

Cloudflare introduces dynamic workers that spin up isolated sandboxes for AI agents in milliseconds. Each agent action runs in its own container with no cross-contamination.

As agents get more autonomous, sandboxing goes from nice-to-have to essential. This makes it cheap and fast.

🔒

Ente's local LLM app — private AI on your own hardware

The privacy-focused team behind Ente (encrypted photo storage) ships a local LLM app. Everything runs on-device, nothing phones home. Classic Ente vibes.

A credible privacy-first team building local AI tools — worth keeping on your radar if you care about data sovereignty.

🛡️

Package managers need dependency cooldowns — here's how

After the LiteLLM supply chain attack (a credential stealer hidden in a PyPI package), Simon Willison revisits dependency cooldowns — waiting a few days before installing new versions so the community can spot compromises.

Supply chain attacks are the quiet crisis in AI tooling. Cooldowns are a simple, practical defense you can adopt today.

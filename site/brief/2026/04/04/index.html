New Tools & Things You Can Try

Apfel: Free AI Already on Your Mac

Popular on HN — Apfel is a Mac app that gives you free access to AI models you already have installed locally. No subscriptions, no API keys, no sending data anywhere. If you've been hesitant to try local inference, this is a dead-simple entry point. Works with open-source models like Llama and Mistral.

Claude Code Found a Linux Vulnerability Hidden for 23 Years

This is the story that's dominating the vulnerability research world right now. Claude Code spotted a real security flaw in the Linux kernel that humans missed for over two decades — and it did it by reading through old code comments. Simon Willison's take captures the bigger picture: frontier models have fundamentally changed the game for security researchers. HN thread.

Anthropic Bans OpenClaw, Makes It Expensive for Claude Users

Starting today, Anthropic is making it much harder to use OpenClaw (a popular AI agent tool) with Claude on paid plans. Users will need to use a different model or pay extra. This is a notable move: Anthropic essentially deciding that OpenClaw posed enough of a security or policy risk that they'd restrict access rather than ban it outright. Research & Thinking

Cognitive Surrender: AI Users Abandon Logical Thinking

New research shows that when people use AI to answer questions, they're surprisingly willing to accept incorrect answers without fact-checking. Large majorities in experiments just... trusted the AI and moved on. It's the kind of finding that should make product builders think hard about how to surface confidence levels, uncertainty, and sources. Not doom — just a design challenge.
